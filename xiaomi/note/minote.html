css样式
所有的css样式一般情况下都写到后缀为.css的文件中


1.css命名：两种方式：
	class: 可以重复的使用      .clss名{具体的样式表}
	id:  唯一的，一个网页里名字只能出现一次      #id名{具体的样式}

2.引入外部文件 用link标签
   	
	type属性：text/css-->引入的文件是一个文本格式css的文件
	href属性："这个文件引入路径"
	rel属性：样式表
3.选择器
  1）id选择器：
  2）class选择器
  3） #Pelem,#Pelem1{}   群组选择器
  4）*-->通用选择器(代表了所有的标签)
	 *{      
		padding: 0;      //重置浏览器
		margin: 0;
	  }
    浏览器里紫色的地方---》padding的值
 浏览器里黄色色的地方  ---》margin的值
 
  5）选择器：hover ---> 伪类选择器   ---》鼠标移上去样式
  
      a:hover{   
  			
      }

	    标签{
	    	
	    }
	    
	  ul li{
	  	
	  }

	  }

    width:   宽

    background:  背景
    float: left  /right  浮动   （向左浮动   向右浮动）
    color: 给文件添加颜色
    margin: 0 auto;  --->div块水平居中

    list-style: none; 去掉li 的前面点

    margin:   外边框
    padding与margin的取值一样
	    padding:10px
	        如果值为一个：上下左右都是10px
	   padding:10px 15px
	       如果值为两个：上下  左右
	   padding:10px 20px 30px 40px;
    			上      右         下        左
       padding-top:
       padding-left:
       padding-right:
       padding-bottom: 
    
    	margin-top:
       margin-left:
       margin-right:
       margin-bottom: 
    
    
    绝对定位：position: absolute;   /*绝对定位*/
		left: 10px;
		top: 10px;
		相对于父级
		
  1.给父级写position:relative
  2.给子级写position：absolute
          left:
          top: 
          控制子级的位置，活动范围是父级别元素
          不给父元素添加position:relative ,子元素的活动范围是body
